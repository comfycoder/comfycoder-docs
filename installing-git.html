Installing Git

Navigate to the Git home page.
https://gitforwindows.org/

Click the Download button to obtain the Git setup program.


Run the Git Setup program.

The Git setup wizard appears.


Click the Next button.

The Select Components step appears.


Click the Next button.

The Choose Editor step appears.


Select your favorite editor from the dropdown list. (e.g., Visual Studio Code)

Click the Next button.

The Path view appears.


Click the Next button.

The Choose Https Transport step appears.


Recommend chooseing the Use the OpenSSL library option.

Click the Next button.

The Configure Line Endings view appears.


Recommend choosing the Checkout Windows-style, commit Unix-style line endings option.

Click the Next button.

The Configure Terminal Editor view appears.

Recommend choosing the Use MinTTY option.

Click the Next button.

The Configure Extra Options view appears.

Click the Next button.

The Git installation commences.

The Git installation completes.

Click the Finish button.
