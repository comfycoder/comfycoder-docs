Installing Azure CLI

The Azure CLI is a command-line tool providing a great experience for managing Azure resources. The Azure CLI is designed to make scripting easy, query data, support long-running operations, and more.

Navigate to the following link:
https://docs.microsoft.com/en-us/cli/azure/install-azure-cli-windows?view=azure-cli-latest

Click on the Download the MIC Installer button.

The browser downloads the installer to your local computer.

Run the Azure CLI setup program.

The Azure CLI setup wizard appears.

When the wizard has initialized, click the Next button.

The Azure License Agreement view appears.

You must click the I accepts the terms in the License Agreement checkbox.

Click the Install button.
