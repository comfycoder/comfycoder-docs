Installing Docker Desktop For Windows

Docker is a tool designed to make it easier to create, deploy, and run applications by using containers. Containers allow a developer to package up an application with all of the parts it needs, such as libraries and other dependencies, and ship it all out as one package. By doing so, thanks to the container, the developer can rest assured that the application will run on any other Linux machine regardless of any customized settings that machine might have that could differ from the machine used for writing and testing the code.

Enable Microsoft Hyper-V

Microsoft Hyper-V is a native hypervisor that can create virtual machines on x86-64 systems running Windows.

Prior to installing Docker Desktop, you need to enable Hyper-V on Windows 10.

Open a PowerShell Command line window as an administrator.

The PowerShell Command line appears.

Run the following PowerShell command statement:

Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Hyper-V -A

Windows prompts you with the following:

Type “Y” and press the Enter key.

Windows run the PowerShell command and enables Hyper-V.

Install Docker

Navigate to the Docker getting-started page.
https://www.docker.com/get-started

Scroll down the page until you see the Download for Windows button, and click on it.

The Docker Desktop Windows download page appears.

Scroll down the page util you see the following:

Click the Get Docker Desktop - Windows (stable) button.

Your browser downloads the Docker for Windows Installer program.

Run the Docker for Windows Installer program.

The Docker for Windows Installer program appears.

If you are going to be creating linux container, do not check the Use Windows containers… checkbox.

Click the Ok button.

The Docker for Windows Installer program commences installing Docker Desktop on your system.

When the installation successfully completes, the following view appears.

Click the Close button.

You are now ready to configure Docker Desktop.

Login To Docker Hub

The first time you run Docker Desktop, it prompts you to login with your Docker ID.

Enter your Docker user account ID and password.

Click the Login button.

Configure Docker Settings

From the Windows 10 Task Bar, open the system tray icons, and right-click on the docker icon.

The Docker context menu appears.

Click on the Setting menu item.

The Docker Setting dialog appears.

Click on the Shared Drives sidebar menu item.

The Shared Drives view appears.

Check the box next to the “C” drive.

Click the Apply button.

Docker will now be able to access to your “C” drive with any container you run.
