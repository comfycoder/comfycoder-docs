Installing Docker Desktop For Windows

Navigate to the Docker getting-started page.
https://www.docker.com/get-started

Scroll down the page until you see the Download for Windows button, and click on it.

The Docker Desktop Windows download page appears.

Scroll down the page util you see the following:

Click the Get Docker Desktop - Windows (stable) button.

Your browser downloads the Docker for Windows Installer program.

Run the Docker for Windows Installer program.

The Docker for Windows Installer program appears.

If you are going to be creating linux container, do not check the Use Windows containers… checkbox.

Click the Ok button.

The Docker for Windows Installer program commences installing Docker Desktop on your system.

When the installation successfully completes, the following view appears.

Click the Close button.

You are now ready to configure Docker Desktop.

From the Windows 10 Yask Bar, open the system tray icons, and right-click on the docker icon.

The Docker context menu appears.

Click on the Setting menu item.

The Docker Setting dialog appears.

Click on the Shared Drives sidebar menu item.

The Shared Drives view appears.

Check the box next to the “C” drive.

Click the Apply button.

Docker will now be able to access to your “C” drive with any container you run.
